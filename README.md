# Solitude
Simple programming language for beginners.

## Syntax
### Variables
```solitude
var x=5
```
### Print
```solitude
Hello World

var x=5
$x
```

### If
```solitude
if x>5
x is greater than 5
```

### Functions
```solitude
func greet Hello!
call greet
```

### Comments
```solitude
. I am a comment
```

### User Input
```solitude
input x Enter a number for x:
$x
```
